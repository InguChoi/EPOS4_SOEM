BUILDING
========


Prerequisites
-------------------------------

 * CMake 3.9 or later


Linux & macOS
--------------

   * `mkdir build`
   * `cd build`
   * `cmake ..`
   * `make`


Documentation
-------------

See https://openethercatsociety.github.io/doc/soem/
